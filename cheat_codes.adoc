== Quote a word, using single quotes:
```
ciw'Ctrl+r"'
```

* `ciw` - Delete the word the cursor is on, and end up in insert mode.
* `'` - add the first quote.
* `Ctrl+r"` - Insert the contents of the " register, aka the last yank/delete.
* `'` - add the closing quote.
